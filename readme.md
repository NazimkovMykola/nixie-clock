# Годинник на лампах ИН-14 
Перед початком роботи виставити 170В на модулі живлення ламп.


## Як підключити піни до Arduino (згідно з цим кодом):
Катоди (Цифри 0-9): Піни D6, D7, D8, D9, D10, D11, D12, D13 A2, A3

Аноди (Лампи 1-4): Піни D2, D3, D4, D5 (використовуються як цифрові).

RTC (DS3231): SDA -> A4, SCL -> A5.

## Керування катодами цифр
Всі виводи цифр ламп (окрім ком, які розташовані по боках від аноду) з'єднуємо між собою. Транзистори MPSA42 (10шт по 1шт на кожну цифру) підєднуємо по схемі: 
* емітер - на загальний "мінус"
* база - через резистор 1кОм до відповідного піну Arduino
* колектор - на відповідний катод лампи

## Анодні ключи (4шт, по кількості ламп)
* MPSA42 - емітер на "мінус", база через резистор 1кОм на керуючий пін Arduino, колектор - на базу MPSA92 через резистор 470кОм
* MPSA92 - емітер - +170В, база - до колектору MPSA42, колектор - на анод лампи через резистор 22кОм. Між базою та емітером резистор 68кОм.

## Принцип роботи годинника
Керування газорозрядними індикаторами (Ніксі-лампами) у коді реалізовано за принципом динамічної індикації. Оскільки Arduino має обмежену кількість виходів, а кожна лампа потребує 10 катодів (цифр) та 1 анод, використання прямого підключення для 4 ламп вимагало б 44 піни. Динамічна індикація дозволяє обійти це обмеження.

Ось як це працює:

1. Спільна шина катодів (Цифри)
Усі ваші лампи з'єднані між собою паралельно по цифрах.
Масив cathodePins[] містить 10 пінів, що відповідають цифрам від 0 до 9.
Якщо подавати сигнал на cathodePins[5], цей сигнал приходить на цифру "5" одразу в усі чотири лампи.

2. Роздільні аноди (Вибір лампи)
Кожна лампа має свій індивідуальний анод, підключений до масиву anodePins[].
Лампа світиться лише тоді, коли одночасно подано напругу і на її анод, і на один із катодів.
Це дозволяє "вмикати" лампи по черзі, дуже швидко перемикаючись між ними.

3. Алгоритм у функції displayDigit
Ця функція виконує один крок відображення для конкретної лампи:
Вибір цифри: digitalWrite(cathodePins[digit], HIGH); — готуємо потрібну цифру на спільній шині.
Активація лампи: digitalWrite(anodePins[lampIndex], HIGH); — подаємо живлення на конкретну лампу.
Пауза: delay(2); — тримаємо лампу увімкненою 2 мілісекунди. Людське око не помічає перемикання, якщо воно відбувається швидше ніж 50-60 разів на секунду (ефект інерції зору).
Вимкнення: Обидва сигнали знімаються, щоб уникнути "привидів" (слабкого світіння сусідніх цифр) при переході до наступної лампи.

4. Цикл у loop
Програма постійно "пробігає" по колу:
Показує десятки годин на 1-й лампі (2 мс) -> Вимикає.
Показує одиниці годин на 2-й лампі (2 мс) -> Вимикає.
Показує десятки хвилин на 3-й лампі (2 мс) -> Вимикає.
Показує одиниці хвилин на 4-й лампі (2 мс) -> Вимикає.
