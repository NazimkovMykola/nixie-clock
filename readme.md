# Годинник на лампах ИН-14 

## Як підключити піни до Arduino (згідно з цим кодом):
Катоди (Цифри 0-9): Піни D2, D3, D4, D5, D6, D7, D8, D9, D10, D11.

Аноди (Лампи 1-4): Піни A0, A1, A2, A3 (використовуються як цифрові).

RTC (DS3231): SDA -> A4, SCL -> A5.

# Номінали радіодеталей
* +180 на Анод через резистор 22кОм
* резистор між базою та емітером MPSA92 68кОм
* резистор 1кОм - до пінів Ардуіно
* резистор 470кОм між колектором та базою MPSA42 та MPSA92
* транзистори MPSA42 для керування катодами цифр
* ключ МPSA42+MPSA92 для керування анодами ламп

## Керування катодами цифр
Всі виводи цифр ламп (окрім ком, які розташовані по боках від аноду) з'єднуємо між собою. Транзистори MPSA42 (10шт по 1шт на кожну цифру) підєднуємо по схемі: 
* емітер - на загальний "мінус"
* база - через резистор 1кОм до відповідного піну Ардуіно
* колектор - на відповідний катод лампи

## Анодні ключи (4шт по кількості ламп)
* MPSA42 - емітер на "мінус", база через резистор 1кОм на під Ардуіно, колектор - на базу MPSA92 через резистор 470кОм
* MPSA92 - емітер - +170В, база - до колектору MPSA42, колектор - на анод лампи через резистор 22кОм. Між базою та емітером резистор 68кОм.
